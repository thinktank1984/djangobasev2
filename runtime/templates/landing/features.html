Django + Django REST Framework

High-level Python web framework that encourages rapid development and clean, pragmatic design.

Tailwind CSS

Utility-first CSS framework for rapidly building custom designs without leaving your HTML.

  • Utility classes for rapid development
  • Highly customizable and extensible
  • Responsive design by default

Alpine.js

Minimal JavaScript framework for adding behavior to your markup with a simple declarative syntax.

  • Declarative and reactive syntax
  • Lightweight (less than 8kb gzipped)
  • Integrates perfectly with Tailwind

HTMX

Library that allows you to access modern browser features directly from HTML, without the need for JavaScript.

  • AJAX, CSS Transitions, WebSockets and Server Sent Events PARTIAL IMPLEMENTATION
  • Small (~14KB min.gz'd)
  • Interactive experiences without writing JavaScript IN DEVELOPMENT
